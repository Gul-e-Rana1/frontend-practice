# frontend-practice

## Project Overview
This project is a practice-based frontend exercise in which I attempted to recreate a layout that resembles an official website design. The purpose of this project is learning, improving UI skills, and practicing responsive website development.

## Disclaimer
This design is not my original creation. I do not claim ownership of the original website's layout, visual identity, or branding. This project is only a personal practice attempt to study and replicate the look and structure of an existing official website for learning purposes.

## Purpose of This Project
- Improve HTML and CSS skills
- Practice recreating modern website layouts
- Understand responsive design behavior
- Learn the structure of professionally made UIs
- Experiment with hero sections, navigation menus, and styled components

## Technologies Used
- HTML5
- CSS

## Features Practiced
- Responsive navigation bar
- Hero section with badge and social icons
- Footer layout practice
- Hover effects and clean UI layouts

## Note
This project is created only for learning and practice. It is not intended for commercial use and should not be considered an official or original design.
